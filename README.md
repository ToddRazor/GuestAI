# Подсчет количества гостей за столом в кафе

Проект выполнен в рамках производственной практики по дисциплине  
«Системы искусственного интеллекта».

## Тема проекта

Разработка веб-приложения для автоматического подсчета количества гостей за столом в кафе с использованием предобученной модели YOLOv8.

## Описание проекта

Веб-приложение позволяет пользователю загрузить изображение кафе или стола с гостями.  
После загрузки изображение обрабатывается предобученной нейросетевой моделью YOLOv8.

Система определяет людей на изображении, отрисовывает рамки вокруг найденных объектов и выводит количество обнаруженных гостей.

## Функциональность

В приложении реализованы следующие возможности:

- загрузка изображения через веб-интерфейс;
- обработка изображения с помощью YOLOv8;
- обнаружение людей на изображении;
- отрисовка bounding boxes вокруг найденных гостей;
- подсчет количества гостей;
- вывод времени обработки;
- сохранение истории обработок в базу данных SQLite;
- выгрузка истории обработок в Excel;
- очистка истории и сохраненных изображений.

## Используемая модель

В проекте используется предобученная модель:

```text
yolov8s.pt
```

Модель применяется для задачи детекции объектов.  
Для подсчета гостей используется только класс `person`.

Параметры обработки:

```python
classes=[0]
conf=0.35
iou=0.45
```

## Запуск проекта

1. Склонировать репозиторий или скачать проект.

2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Запустить приложение:

```bash
python app.py
```

4. Открыть приложение в браузере:

```text
http://127.0.0.1:5000
```
